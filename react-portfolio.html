Portfolio Redesign as React SPA

Project introduction

A large project that I am currently working on, where I am to completely convert my static portfolio website into a fully fledged React single-page application.

Progress is still ongoing and isn't hosted online yet. I am happy to provide a live preview upon request. See ongoing progress on GitHub

Project aims:

  • Completely rebuild existing static portfolio as a React SPA.
  • Incorporate dynamic rendering and reusability of components.

Project progress so far (as of 20th May 2024)

  • Converted navigation system into dynamic component and sub-components.
  • Set up dynamic image sourcing using createContext React hook, and wrapping all components inside.
  • Created dynamic components and sub-components for hero, skills, certifications and career slices respectively.
#

Roadmap for the rest of the project

  • Continue refactoring existing sections into dynamic components.
  • Make sidewide JS local to its respective components.
