Production of a design-phase accessibility compliance checklist

Project introduction

A smaller project, in which I produced an accessibility checklist to be used by designers. The goal of this checklist is to ensure that design work is compliant with the relevant WCAGs and that as few accessibility slip through into the development phase as possible.

A checklist was produced and made easily accessible for designers, which is currently being used successfully.

Project aims:

  • Determine which WCAGs are relevant to the design phase.
  • Determine to which level the selected WCAGs need to be met.
  • Determine how these WCAGs must be met, and how the compliance can be measured as a pass/fail.
  • Produce a checklist containing all of the selected WCAGs, organised into categories and within reach, so the checklist can be used as quickly and easily as possible.
  • Include other suggested resources that designers can use to improve accessibility of their produced UI designs.
  • Checklists to be attached to Jira issues to demonstrate that they have been audited and are compliant.

The finished product(s)

Checklist

  • Checklist was produced containing all the relevant WCAGs, organised into categories.
  • Created as an XD library component, meaning easy dragging-and-dropping into project.
  • Checklist also placed centrally on Confluence within a new "Accessibility Tools" section, for easy access by all.
  • Pass/fail status can be selected easily by deleting the non-applying option.

Accessibility tools bank

  • A bank of several suggested tools and plugins, to help designers to achieve higher levels of accessibility in produced UI design work.
  • Also added to the XD library component, meaning easy dragging and dropping into project.
  • Also placed centrally on Confluence within a new "Accessibility Tools" section, for easy access by all.
  • Tools includes a suggested accessibility XD plugin, drag and drop mobile reach map to inform placement of important elements and WebAim colour contrast checker.
  • New tools are being considered and will be added to bank over time.
