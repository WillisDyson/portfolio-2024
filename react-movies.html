React Movies Search with API, Dotenv & Webpack

Project introduction

A small project in which I showcase my React skills. I use React to create a Movies Search tool, which returns movies based on the user's input through a GET request. The tools uses Dotenv/Webpack to hide the API key from public access.

Project aims:

  • Create a "movie search" functionality in React, which links to the TMDB database through an API.
  • Movies must be returned via GET request, and must be updated without rerendering the whole component.
