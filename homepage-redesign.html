TopCashback homepage offers redesign

Project introduction

A smaller project in which the aim was to modernise and enable accessibility for TopCashback's homepage offer cards.

My roles within the project:

  • Performed initial testing and analysis to highlight problems with the current offer cards.
  • Produced new designs for improved offer cards, using the latest brand fonts/styles and incorporating rounded corners.
  • Ran impact analysis discussions with Heads of different departments, to ensure that all angles had been considered and that the change had been fully prepared for.
  • Built the improvement, ensuring that the responsiveness issues have been resolved, and that offer card content is fully accessible.
  • Performed cross-device and cross-browser testing.

Legacy homepage offer cards

Problems

  • Very outdated design, with sharp edges and design/layout inconsistencies.
  • Not enough white space, meaning limited visual focus allowed for the cashback rate.
  • Responsiveness issues, with the offer cards becoming very small on some mobile devices.
  • Inefficient layout method, confined to using "floats" due to Bootstrap grid implementation.

New homepage offer cards

Improvements

  • Modernised design with increased padding and rounded corners.
  • Smaller card image, with more visual focus on the cashback description and rate.
  • Greatly-improved responsiveness. Moved from Bootstrap to CSS Grid, with cards now responding properly and not decreasing in size unnecessarily at mobile screen widths.
  • Much more up-to-date with the latest brand styles.
